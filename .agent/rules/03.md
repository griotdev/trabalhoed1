# Comandos de Entrada (.geo e .qry)

## Arquivo .geo (Geometria Inicial)
[cite_start]Define as formas que estão inicialmente espalhadas no "chão"[cite: 367].

| Comando | Parâmetros | Descrição |
| :--- | :--- | :--- |
| `c` | `i x y r corb corp` | [cite_start]Cria Círculo (id, x, y, raio, cor borda, cor preenchimento)[cite: 366]. |
| `r` | `i x y w h corb corp` | [cite_start]Cria Retângulo (id, x, y, w, h, cor borda, cor preenchimento)[cite: 366]. |
| `l` | `i x1 y1 x2 y2 cor` | [cite_start]Cria Linha (id, x1, y1, x2, y2, cor)[cite: 366]. |
| `t` | `i x y corb corp a txt` | [cite_start]Cria Texto (id, x, y, cores, âncora, string texto)[cite: 366]. |
| `ts` | `family weight size` | [cite_start]Define estilo de fonte para os próximos textos[cite: 366]. |

## Arquivo .qry (Consultas e Jogo)
Manipula as peças, disparadores e realiza o processamento do jogo.

| Comando | Parâmetros | Descrição |
| :--- | :--- | :--- |
| `pd` | `id x y` | [cite_start]Posiciona um Disparador (id) na coordenada (x,y)[cite: 375]. |
| `lc` | `id_carregador n` | [cite_start]Carrega (`Load Charger`) as primeiras `n` formas do chão para o carregador[cite: 375]. |
| `atch` | `id_disp id_cesq id_cdir` | [cite_start]Acopla (`Attach`) carregadores (esq/dir) a um disparador[cite: 375]. |
| `shft` | `id_disp [e|d] n` | [cite_start]Manipula as pilhas internas do disparador (veja lógica detalhada)[cite: 375]. |
| `dsp` | `id_disp dx dy [v]` | [cite_start]Dispara a forma na posição de tiro com vetor de deslocamento (dx, dy)[cite: 375]. |
| `rjd` | `id_disp [e|d] dx dy ix iy` | [cite_start]Rajada: executa `shft` de 1 item seguido de `dsp` repetidamente até esvaziar carregador[cite: 375]. |
| `calc` | (sem parâmetros) | [cite_start]Processa a colisão/pontuação das formas lançadas na arena[cite: 375]. |
